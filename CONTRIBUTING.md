# Contributing to Adopti 🚀

Thank you for your interest in contributing to **Adopti** — an AI-powered marketing analytics platform.

We welcome contributions from developers, designers, and data enthusiasts who want to help improve the platform.

---

## 📌 About the Project

Adopti is a SaaS platform that:

* Connects marketing platforms (Google Ads, etc.)
* Tracks campaign performance
* Provides dashboards and analytics
* Generates AI insights and recommendations

---

## 🛠️ How to Contribute

### 1. Fork the Repository

Click the **Fork** button and clone your fork:

```bash
git clone https://github.com/sahilnagwani/adopti.git
```

---

### 2. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

Examples:

* `feature/dashboard-improvement`
* `fix/metrics-calculation`
* `feature/ai-recommendations`

---

### 3. Make Your Changes

Please ensure:

* Code is clean and readable
* Follow existing project structure
* Do not break existing functionality
* Use consistent naming conventions

---

### 4. Test Your Changes

Before submitting:

* Ensure app runs without errors
* Check dashboard, campaigns, and analytics pages
* Verify Supabase queries still work correctly

---

### 5. Submit a Pull Request

* Push your branch
* Open a Pull Request (PR)
* Clearly describe:

  * What you changed
  * Why it was needed

---

## 🐛 Reporting Issues

If you find a bug or want to suggest a feature:

1. Go to **Issues**
2. Click **New Issue**
3. Provide:

   * Clear description
   * Steps to reproduce (if bug)
   * Expected vs actual behavior

---

## 💡 Contribution Ideas

You can contribute by working on:

* Dashboard UI improvements
* Performance optimization (queries, caching)
* AI insights & recommendations logic
* Data visualization (charts)
* Integrations (Google Ads, Meta Ads)
* UX improvements (loading states, responsiveness)

---

## ⚠️ Guidelines

* Always respect multi-tenant logic (`workspace_id`)
* Do not modify database schema without discussion
* Avoid introducing breaking changes
* Keep the app scalable and modular

---

## 🤝 Code of Conduct

Be respectful and collaborative. We aim to build an open and supportive community.

---

## ⭐ Support the Project

If you like Adopti:

* Star the repo ⭐
* Share it with others
* Contribute improvements

---

Thanks for helping make Adopti better! 🚀
