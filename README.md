# 🚀 Adopti — AI Marketing Analytics Platform

Adopti is an AI-powered marketing analytics SaaS platform that helps businesses track campaign performance, generate insights, and make smarter optimization decisions.

---

## ✨ What Adopti Does

Adopti connects marketing data (e.g., Google Ads), aggregates performance metrics, and transforms them into:

* 📊 Unified dashboards
* 📈 Campaign analytics
* 🤖 AI-generated insights
* ⚡ Optimization recommendations

---

## 🧠 Core Features

### 📊 Dashboard

* Track Spend, Revenue, ROAS, Conversions
* Visualize trends with charts
* Identify top-performing campaigns

---

### 📢 Platform Analytics

* Deep dive into platform-level performance
* Campaign comparison
* Trend analysis

---

### 🎯 Campaign Analytics

* Campaign-level breakdown
* Daily performance tracking
* ROAS, CTR, CPA insights

---

### 🤖 AI Insights

* Best & worst performing campaigns
* Trend analysis
* Performance explanations

---

### ⚡ AI Recommendations

* Budget scaling suggestions
* Low-performance alerts
* Optimization opportunities

---

## 🧱 Tech Stack

**Frontend**

* React
* Tailwind CSS

**Backend**

* Supabase (PostgreSQL)
* Row Level Security (multi-tenant)

---

## 🗄️ Architecture

Adopti follows a scalable SaaS architecture:

Ad Platforms → Data Layer → Supabase DB → Service Layer → React UI → AI Insights

---

## 🔐 Multi-Tenant System

* Workspace-based data isolation
* Secure access using RLS
* Supports teams and collaboration

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/your-username/adopti.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Add environment variables

```
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```

### 4. Run the app

```bash
npm run dev
```
